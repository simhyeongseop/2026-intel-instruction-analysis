# 1회 미팅 내용

**날짜**: 2026-03-10

## 📌 미팅 주제

Intel x86/x64 어셈블리 명령어 추출 프로젝트 킥오프

---

## 🎯 주요 논의 사항

### 1. 프로젝트 배경

- 어셈블리 명령어의 기본 구조 이해
- PDF 매뉴얼에서 명령어 신텍스 추출의 필요성
- 컴파일러/어셈블러의 정확성 검증

### 2. 1단계: 명령어 리스트 추출

**목표**: Intel 매뉴얼(Volume 2)에서 A~Z 명령어 신텍스 추출

**방식 선택**:
1. **PDF TO TEXT** 변환
   - 간단하지만 포맷 손실 위험
   
2. **PDF LIBRARY** 활용 (선택됨)
   - Python의 pdfplumber 사용
   - 테이블 구조 파싱
   - 더 정확한 데이터 추출 가능

3. **REGULAR EXPRESSION**
   - 패턴 매칭을 통한 명령어 필터링
   - `[A-Z0-9]*` 형태 감지

### 3. 기술적 조언

**Python 활용**:
- pdfplumber 라이브러리로 PDF 테이블 파싱
- 정규표현식(Regex)을 활용한 명령어 검증
- 예외 처리(Exception Handling) 중요

**학습 팁**:
- 전문가를 위한 Python 책 추천
- 디버깅 방법 (PDB) 숙지
- 작은 예제부터 시작하여 단계적 확장

### 4. 명령어 구조 이해

- **Mnemonic**: 명령어 이름 (ADD, MOV, etc.)
  - 보통 대문자로 시작
  - 2글자 이상의 알파벳 + 숫자 가능
  
- **Opcode**: 기계어 코드
  - CPU가 이해하는 바이너리 형식
  
- **Operand**: 피연산자 (데이터, 레지스터)
  - r16 (16비트 레지스터)
  - m16 (16비트 메모리)
  - imm8 (8비트 상수값)
